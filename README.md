# Hash Table in C

Implemented Hash table in C using dbj2 for string hash.

The shakespeare text comes from [this](https://ocw.mit.edu/ans7870/6/6.006/s08/lecturenotes/files/t8.shakespeare.txt).

## Build and run

```
cc -o cht cht.c
./cht ./t8.shakespeare.txt
```
